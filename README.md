# Python & Data Science Core Learning Journey 🚀

Welcome to my professional learning portfolio! This repository is a systematic, day-by-day track of my journey into Data Science and Python Development. It contains clean scripts, Jupyter Notebooks, and practical implementations of data wrangling, exploration, and visualization techniques.

---

## 📅 Roadmap & Learning Milestones

Here is the structured breakdown of the concepts and core skills I have mastered so far:

### 🔹 Phase 1: Python Basics & Environment Setup

* **`Day-02-Python-Scripting-and-Notebooks`**
  * Setting up development environments inside VS Code.
  * Working with Python scripts (`.py`) and Jupyter Notebooks (`.ipynb`).
* **`Day-03-Python-Basics-and-Core-Concepts`**
  * Writing the first lines of executable code.
  * Core fundamentals: Operators, Strings, Comments, Variables, and user Input fields.
  * Implementing Conditional Logics (`if-elif-else` structures).
* **`Day-05A-Environments-and-Libraries`**
  * Managing virtual environments.
  * Downloading, installing, and calling external libraries/packages.
* **`Day-05B-Jupyter-Markdown-Language`**
  * Mastering Markdown language syntax to create beautifully documented, professional notebook reports.

### 🔹 Phase 2: Core Data Wrangling (Pandas & NumPy)

* **`Day-06-Pandas-Basics-and-Data-Exploration`**
  * Importing Pandas, loading external datasets (Excel/CSV matrices).
  * Inspecting data frames using foundational methods: `.head()`, `.tail()`, `.info()`, `.shape`, `.index`, and `.describe()`.
* **`Day-07-Data-Wrangling-and-Sorting-with-Pandas`**
  * Data cleaning basics: stripping whitespaces from structural columns.
  * Advanced manipulation: Typecasting features, transposing structures, and sorting datasets via `.sort_index()` and `.sort_values()`.
* **`Day-10-Ten-Minutes-to-Pandas-and-Numpy-Core`**
  * Deep dive into the official "10 Minutes to Pandas" curriculum alignment.
  * Understanding the underlying mathematical engine (NumPy) driving structural computations.

### 🔹 Phase 3: Advanced Data Analysis & Visualizations

* **`Day-08-Advanced-Pandas-Tricks-and-Data-Visualization`**
  * Working with standard benchmark datasets (Titanic & Iris).
  * Advanced tricks: generating random dataframes, reversing row/column orders, categorical-to-continuous conversion, joining/splitting subsets, and group aggregations.
* **`Day-11-Data-Visualization-with-Seaborn-and-Matplotlib`**
  * Translating raw numeric features into data stories.
  * Implemented: Scatter plots (with `hue` semantic mapping), Box plots for outlier detection, Histograms with Kernel Density Estimates (`kde=True`), and customized axes/labels.
* **`Day-12-Exploratory-Data-Analysis-and-Data-Cleaning`**
  * Full-cycle Exploratory Data Analysis (EDA) & Imputation.
  * Diagnosing missing data via `.isnull().sum()`, dropping invalid features, handling duplicates, and filling values using statistical averages (Mean imputation).
* **`Day-13-Feature-Correlation-and-Heatmaps`**
  * Isolating numeric features to perform mathematical statistical analysis.
  * Generating Correlation Matrices via `.corr()` and visualizing deep patterns using annotated Heatmaps (`annot=True`, `cmap='coolwarm'`).
